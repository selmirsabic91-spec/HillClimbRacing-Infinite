# Hill Climb Racing - Infinite Fuel Edition

Android igrica sa beskonačnim benzinom - Hill Climb Racing klone za Android.

## 🎮 Karakteristike

✅ **Beskonačan benzin** - Vozite bez brige o gorivanju
✅ **Physics engine** - Realistična fizika automobila
✅ **Raznovrsni tereni** - Brežuljci, planine, ceste
✅ **Nadogradnje vozila** - Poboljšajte automobil
✅ **Offline igranje** - Bez potrebe za internetom
✅ **Android optimizovano** - Radi na svim verzijama

## 📋 Zahtevi

- Android 5.0+ (API 21+)
- 100 MB memorije
- Omogućeni senzori (accelerometer)

## 📥 Instalacija

### BRZO - Preuzmi APK direktno:

1. Idi na **Releases** → Preuzmi `app-release.apk`
2. Otpremite APK na Android telefon
3. Klikni na APK i instaliraj
4. IGRAJ!

### ILI - Build sam:

```bash
git clone https://github.com/selmirsabic91-spec/HillClimbRacing-Infinite.git
cd HillClimbRacing-Infinite
./gradlew assembleDebug
adb install app/build/outputs/apk/debug/app-debug.apk
```

## 🎮 Igranje

- **DESNO (Gas pedal)** - Pritisnite desni dio ekrana za ubrzanje
- **LEVO (Brake pedal)** - Pritisnite levi dio ekrana za kočenje
- **PAUSE** - Pauziraj igru
- **RESTART** - Ponovna igra
